Set a new size for this GameObject. Current size before change does not affect new size.

Change this object size. The new object size is the sum of the current size plus the values provied.

Set whether an object may be teleported to by the player. Objects are set to be teleportable by defaults when they are created.
setObjectText
Set text to display directly above this object. If the text to display is long, the text height and font size may need to be adjusted in the methods: setObjectTextHeight and setObjectTextFontSize.
setObjectTextFontSize
Set the font size of text set above this object. The default fonot size if 5.
setObjectTextHeight
Set the height that text set on this object will sit above the object. The height value specified is relative to the current height (default .5 above the top of the object).
setObjectThrowable
Set whether this object may be grabbed and thrown. An object set as throwable has gravity set by default.
setObjectVelocity(Vector3)
Set velocity for this object
setObjectVelocity(Single, Single, Single)
Set velocity for this object
setObjectVisible
Set whether this object is visible or not
setPlayerPositionAbsolute
Set a new player position (not relative to current position)
setPlayerPositionRelative
Set the player position relative to the player's current position
setTetherObjectToLeftController
Sets or unsets an object to tether to the left controller. A tethered object will move with the tilt movements of the controller. Tilting the controller up/down (x-axis) will move the tethered object to/from the player. Tilting the controller right/left (z-axis) will move the tethered object orthogonal (perpendicular) to the player.
setTetherObjectToRightController
Sets or unsets an object to tether to the right controller. A tethered object will move with the tilt movements of the controller. Tilting the controller up/down (x-axis) will move the tethered object to/from the player. Tilting the controller right/left (z-axis) will move the tethered object orthogonal (perpendicular) to the player.
startTimer
Start a timer for this object, the length of the timer specified in the argument. To check if the timer is finished, use the isTimerFinished() method.
takePoints
Remove points from the player in specified amount
winGame
Enable win game scenario. Sets win game text on the screen and resets the current scene.
