Moving Files to VR Computer

The files on the students’ computers will need to be moved to the computers with the VR headsets. To do this, several files will need to be copied from the student laptop to a thumb drive, then from the thumb drive to the VR computer.

Files to Copy:

  1. The scene file
  2. The customScripts folder (Assets->API->UWBSummerCampAPI->customScripts)
  3. The _customPrefabs folder (Assets->Resources->_customPrefabs)
  1. This is only necessary if custom prefabs have been made for the game

Copy files to the same folders on the VR computer. Overwrite any files if necessary.
